# PAquant Agents

This directory stores file-based strategy source material for PAquant AI traders.

- `common/price-action-core.md`: shared Brooks price action concepts.
- `common/risk-control.md`: shared risk, order, stop, target, and audit rules.
- `traders/*.md`: one personality and strategy file per registered trader.

Current registered strategy traders cover Brooks Generalist, Always-In Trend, Second Entry,
Best-Trades-Only, Trading Range Scalping, Breakout Pullback, Wedge Reversal,
Breakout Failure, Major Trend Reversal, and Final Flag setups.

The files are public-safe strategy notes. They do not contain API keys, broker credentials, or raw PDF excerpts.
